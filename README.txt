Este programa sirve para sacar el reporte ingresando de manera dinamica la fecha apartir de la cual queremos obtener los registros.

Ejemplo:
Si coloco que me traija los datos apartir del 20 de agosto del 2919 a las 00:00:00 me traera todos los datos que se encuentren a partir de esa fecha para delante 

Tiene comentada las variables de rango mayor para incorporarlo por rangos en vez de sólo declarar el inicio. Pero por la forma en la que hice este programa es problable que cambia de lenguaje a uno que sea más dinamico.
